Primeiros passos no CSS:

Neste exercicio vamos ver algumas tags do CSS, mas primeiro precisamos saber como ele se estrutura

01 - Aprendendo estruturas do CSS

O CSS é dividido em 3 estruturas por ordem de prioridade:

1.1 - Estrutura inline:

Estrutura inline consiste em adicionar os estilos linha por linha de seus codigos.

1.2 - Estrutura interna:

Estrutura interna consiste em adicionar uma tag <style> dentro do head para e englobar os estilo de UMA pagina.

1.3 - Estrutura externa:

Estrutura externa consiste em um link no head que ira adicionar uma pagina propria para configurar o estilo de todas as suas paginas adicionais.Vale criar mais de um link em CSS.

02 - Aprendendo Cores

Existem 4 formas de leitura das cores. São elas:

1.1 - Por nome:

Digitando o nome das cores

Blue, white, Red.

1.2 - Por codigos hexadecimais:

Usando a # e em seguida o codigo hexadecimal da cor desejada.

1,2,3,4,5,6,7,8,9,a,b,c,d,e,f

1.3 - Por RGB:

Usando a tag "rgb" e em seguida colocando a quantidade de red, green, e blue que a cor desejada tem.

1.4 - Por HSL:

Usando a tag hsl e em seguida colocando a quantidade de hue, sturation, e luminosity (Gradação, saturação, e luminosidade) da cor desejada.

03 - Fontes

Existem diferentes tipos de fontes com diferentes tipografias, e diferentes formas de configura-lás.

Adicionando fonte:

Fontes externas:

Para adicionar uma fonta externa, Vá em um site, cole a url da fonte. basta usar a tag: @ + import url, depois adicionar o url da fonte

exemplo: 
                                
@import url('url do site')

Fonte interna:

Para adicionar uma fonte interna, baixamos a fonte no computador e usamos a tag @font-face, e depois o nome exato do arquivo. Podendo adicionar varios formatos de arquivo para garantir o uso em diferentes navegadores.

Exemplo: 
                                
@font-face {
font-family: ;
src: url('nome do arquivo fonte')
format('depende do formato do arquivo'),    
url('nome do segundo arquivo fonte')         
format('depende do formato do arquivo');
}

Obs: Os formatos de arquivo de fonte mais conhecidos são:

  • opentype (otf)
  • truetype (ttf)
  • embedded-opentype
  • truetype-aat (Aplee Advanced Typography)
  • svg

04 - Seletores personalizados

São formas de te ajudar a editar o seu codigo no css. São ela:

Identificação:

Retratados como id no html e # no css é usado para formatação de um elemento unico.

exemplo:

<h1 id="nome escolhido">Lorem ipsum dolor sit, amet consectetur adipisicing elit. Perspiciatis tempora pariatur sint enim veritatis vel iure dolorem itaque, maxime nisi omnis nihil suscipit placeat nemo odit at doloribus cum voluptas!</h1>

Classes:

Retratados como class no html e . no css são usadas para formatação de mais de uma elemento. E um elemento pode ter duas classes.

exemplo:

<h1 class="nome escolhido">Lorem ipsum dolor sit, amet consectetur adipisicing elit. Perspiciatis tempora pariatur sint enim veritatis vel iure dolorem itaque, maxime nisi omnis nihil suscipit placeat nemo odit at doloribus cum voluptas!</h1>

Pseudo-classes ":"

Retratados como ":" na css, são seletores que permitem aplicar animações e estilos as classes.

05 - Caixas

O css/html são configurados basicamente em dois formatos de caixas, veja como usa-las:

Box-Level

São caixas que pulam uma linha do paragrafo e ocupam toda a largura da pagina. Entre as tags box level estão:

  • <div>
  • <h1-h6>
  • <p>
  • <main>
  • <header>
  • <nav>
  • <article>
  • <aside>
  • <footer>
  • <form>
  • <video>

Inline-level:

São caixas que ocupam o espaço necessario, se mantendo dentro do paragrafo. Entre as tags inline-level estão:

  • <span>
  • <a>
  • <code>
  • <small>
  • <strong>
  • <em>
  • <sup e sub>
  • <label>
  • <button>
  • <input>
  • <select>
