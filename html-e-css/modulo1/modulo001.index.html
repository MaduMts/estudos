Principais formatações:

Negrito/destaque:

• Para deixar algo em negrito, se usa a tag: <b>, porem ela é uma tag não semantica, e sim estetica.

• Para deixar em destaque algo semantico (com importancia), se usa a tag <strong>

Italico/ênfase:

• Para deixar algo em italico usamos a tag <i>,porem não é semantico

• Para deixar em ênfase algo semantico (com importancia), se usa a tag <em>

• Dica: para adicionar formatações em locais diferentes, digite Ctrl+shift+P e selecione "emmet: envolver com abreviação" em seguida coloque a tag da formatação desejada e aperte Enter.

Texto marcado:

• Você pode deixar uma parte do seu texto marcado com a tag <mark>

Texto pequeno:

• Você pode deixar seu texto com as letras menores com a tag <small>

Texto inserido e texto deletado:

•Texto inserido: Usado para dar enfase que foi adicionado depois. Basta usar a tag <ins>

•Texto deletado: Usado para indicar que deve ser lido mas não considerado. Basta usar a tag <del>

Sobrescrito e Subscrito

•Sobrescrito: Para escrever coisas tipo: x20+3. Com a tag <sup>

•Subescrito: Para escrever coisas do tipo: H2O. Com a tag <sub>

Citações:

• Para indicar uma sitação simples, porem semantico, usamos a tag <q>

Exemplo:

• Para indicar uma citação completa, e semantica, usamos a tag <blockquote cite=>

Exemplo:
