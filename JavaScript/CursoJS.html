1 - Tipos primitivos em JavaScript:

JavaScript é uma linguagem que tem tipagem dinamica, ou seja, não precisamos declarar o tipo de variavel que estamos usando, o proprio JavaScript faz isso por nós. Porem é necessario entender esses tipos primitivos para que possamos manipular de forma correta. E são eles:
String - Texto
Number - Número
Boolean - Lógico
Null - Nulo
Undefined - Não definido
