1 - Tipos primitivos em JavaScript:

JavaScript é uma linguagem que tem tipagem dinamica, ou seja, não precisamos declarar o tipo de variavel que estamos usando, o proprio JavaScript faz isso por nós. Porem é necessario entender esses tipos primitivos para que possamos manipular de forma correta. E são eles:

String - Texto
Number - Número
Boolean - Lógico
Null - Nulo
Undefined - Não definido
Object - Objeto
Symbol - Simbolo

Vamos ver alguns exemplos de como manipular esses tipos:

1.1 Tipo strings em JavaScript:

Strings basicamente é a manipulação de textos, o padrão o JS entender que tudo é string, por isso muitas vezes não precisamos determinar isso na variavel. Como exemplo usamos o prompt para adicionar uma variavel no site e manipular de varias formas:

1.2 - Tipo number em JavaScript:

Usamos o Number para a manipulação de numeros em geral, e existe algumas formas de trabalhar com eles, aqui vamos ver algumas delas:
Aritmeticos - de ponto flutuante, inteiros
Atribuição - Adição, Subtração, Multiplicação, Divisão, Resto, Exponenciação
Relacionais - >, <,>=, <=,==,===, !=, !==
Lógicos - E &&, OU ||, NÃO !
Ternário - Condição ? Verdadeiro : Falso
Incremento - n += 1, n -= 1 ou n++, n--

1.2.1 - Aritmeticos, Atribuição e Incremento:

Por padrão JS diferencia numeros inteiros de numeros decimais só com a tag "number", mas para confirmar basta usar parseInt para numeros inteiros, e o parseFloat para numeros decimais. Tambem podemos fazer calculos e fazer configurações. Veja os exemplos usando os numeros do prompt:

Vale ressaltar que o JS segue a ordem matemática de prioridade, ou seja, sempre se resolve em ordem:

  • Parenteses ()
  • Exponenciação **
  • Multiplicação * , Divisão / e Resto da divisão %
  • Soma + e Subtração -
  • Maior >, Menor <, Maior igual >=, Menor igual <=, Igual ==, Identico ===, Diferente !=, e Não identico !==
  • Não !
  • E &&
  • Ou ||
  • Condição ? Verdadeiro : Falso
